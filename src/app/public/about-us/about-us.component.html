Who we are

This project is an open source project built by the Vienna Data Science Cafe Meet-Up members.

The goal, except for the obvious result, is to also use this project as a learning hub and as a research playground to build and test different Natural Language Processing models.

The Project status is still under development. You are very welcome to join the team in an upcoming edition of the Vienna Data Science Cafe or participate in the development on GitHub.

The first 50 enrollers will become our alpha testers, so go ahead and register.

You will be notified when things are ready.
